Drawing Tools & Layer Guide

Layer System Basics

What are Layers?

Transparent sheets stacked on top of each other. Each drawing element goes on its own layer for easy editing.

Layer Order

Top layers appear in front. Use Move Up/Down buttons to change stacking order.

Visibility Control

Click the eye icon to hide/show layers. Hidden layers won't appear in your animation.

Essential Drawing Tools

Move Tool

Select, move, rotate and resize objects on layers

  • Click objects to select them
  • Drag to move around canvas
  • Use corner handles to resize
  • Green handle to rotate
Pencil Tool

Draw freehand lines and sketches

  • Click and drag to draw
  • Each stroke creates new layer
  • Adjust brush size in sidebar
  • Change opacity for transparency
Eraser Tool

Remove parts of drawings cleanly

  • Drag over areas to erase
  • Only affects current layer
  • Adjust eraser size as needed
  • Creates transparent holes
Shape Tools

Create perfect geometric shapes

  • Drag to set size and position
  • Each shape on separate layer
  • Primary color for outline
  • Secondary color for fill

Layer Pro Tips

Naming Layers

Double-click layer names to rename them. Use clear names like "Character Head" or "Background Trees"

Grouping Layers

Select multiple layers (Shift+click) then click Group button to organize related elements

Layer Opacity

Click opacity button to cycle through 100%, 75%, 50%, 25% transparency levels

Timing & Spacing Principles

Frame Rates
  • 12 FPS: Traditional animation, choppy but expressive
  • 24 FPS: Smooth cinematic motion
  • 30 FPS: TV standard, very smooth
  • 60 FPS: Ultra-smooth for games
Spacing Rules
  • Ease In: Start slow, accelerate
  • Ease Out: Fast start, slow stop
  • Linear: Constant speed (robotic)
  • Bounce: Overshoot and settle

Advanced Animation Techniques

12 Principles of Animation

1. Squash & Stretch

Deform objects to show impact, weight, and flexibility. A bouncing ball squashes on impact.

2. Anticipation

Prepare the audience for action. A character crouches before jumping.

3. Staging

Direct attention to what's important. Clear silhouettes and composition.

4. Follow Through

Parts continue moving after main action stops. Hair, clothes, tails.

5. Overlapping Action

Different parts move at different rates. Natural, organic motion.

6. Arcs

Most natural actions follow curved paths, not straight lines.

Professional Tips & Tricks

Performance Optimization

Speed Boosters
  • Use lower resolution for rough animation
  • Hide unnecessary layers while working
  • Work in sections, not full scenes
  • Use simple shapes for blocking
Memory Management
  • Delete unused frames and layers
  • Optimize image sizes before import
  • Use groups to organize complex scenes
  • Clear browser cache if sluggish
